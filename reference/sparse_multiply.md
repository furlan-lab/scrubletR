# Multiply each row of a sparse matrix by a scalar

This function multiplies each row of a sparse matrix by a scalar value.

## Usage

``` r
sparse_multiply(E, a)
```

## Arguments

- E:

  A sparse matrix where rows represent observations and columns
  represent features.

- a:

  A scalar value to multiply each row.

## Value

A sparse matrix with each row multiplied by the scalar value.
